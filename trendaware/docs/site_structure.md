# TrendAware: Site Structure

## Pages

### 1. Authentication
- **Login Page** - User login with email/password
- **Registration Page** - New user signup
- **Profile Setup Page** - Initial onboarding workflow

### 2. Dashboard (Home)
- Main landing page after login
- Morning briefing content
- Quick access to all main features

### 3. Themes
- Theme management interface
- Theme creation and editing
- Theme performance metrics

### 4. Content Library
- Saved articles and insights
- Organization and tagging system
- Notes and annotations

### 5. Earnings Calendar
- Calendar view of earnings calls
- Company-specific earnings reports
- Earnings insights and analysis

### 6. Analytics
- Trend visualization dashboards
- Custom insight reports
- Export functionality

### 7. Settings
- User profile management
- Notification preferences
- Integration settings

## Navigation

- **Primary Navigation** - Dashboard, Themes, Library, Calendar, Analytics
- **Secondary Navigation** - Settings, Help, Logout
- **Mobile Navigation** - Condensed menu with core functions
